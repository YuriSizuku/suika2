このソフトについて

  • 自由に使えるノベルゲーム制作ツールです
  • WindowsとMacの両方で動きます
  • ブラウザ版配布キットを使えばブラウザでも動きます
  • iPhone, iPad, Androidでも動作します

インストール/アンインストール

  • ZIPファイルを展開したフォルダ suika2 をそのままご使用ください
  • 不要になりましたらフォルダごと削除してください

サンプルゲームの起動方法

  • Windowsでは、
    • 実際のゲームを起動する場合 suika.exe を開いてください
    • 開発用のツールを起動する場合 suika-pro.exe を開いてください
    • 警告が表示された場合、「詳細情報」をクリックしてから「実行」をクリックしてください
  • Macでは、
    • 実際のゲームを起動する場合、
      • 展開されたフォルダ suika2 を開いてください
      • mac.dmg を開いて、中に入っている suika を、元のフォルダである suika2 にコピーします
      • suika2 の中で、suika を開いてください
    • 開発用のツールを起動する場合、
      • 展開されたフォルダ suika2 を開いてください
      • mac-pro.dmg を開いて、中に入っている Suika2 Pro for Creators を、元のフォルダである suika2 にコピーします
      • suika2 の中で、Suika2 Pro for Creators を開いてください
